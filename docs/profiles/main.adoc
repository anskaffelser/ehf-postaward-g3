== Using this document

This document describes what is needed to make correct configuration of receivers of EHF G3 formats in the Service Metadata Publisher (SMP).


== Profiles

IMPORTANT: Identifiers are strings without spaces. This part of the document contains identifiers where there may be used space to make the identifiers easier to read. *Make sure to remove any spaces in identifiers before use.*
